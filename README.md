# url-shortener
A project to create a url shortener. 

Functions of a Url Shortener
1) Solves the problem of character limits when resharing on certain platforms.
2) It is useful for tracking the users clicking a link.

Features of this Url Shortener
1) Takes a url (Original Url) and returns a shortened version (Short Url).
2) Returns the Original url when the Short Url is inputted.

Additional Features
1) A "Self Destructing Link" that only works once and then deletes itself.
2) Uses regular expressions to ensure strict validation :- that a url is valid and that a url does not end with a .xyz domain.
3) Generates a scannable QR code alongside the short url.
4) A custom HTML "404 Not Found" page for deleted and expired links.
5) Interactive and simplistic form for link submission and retrieval.
